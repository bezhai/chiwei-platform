# 赤尾更像人:下一阶段迭代方向

> 2026-06-13 整理。来源:一轮 agent team 脑暴(9 个"人性维度"各自发散、逐条独立评判),加 bezhai 当场补的关键方向。这是方向蓝图,不是 spec;具体做哪条时再单独写 spec。

## 现在最大的问题:她"做事"是演的,看完得到的东西一眼假

赤尾会说她"在玩手机""在看小说",但**看完能得到什么——观后感、聊起来的内容——全是模型当场编的**。因为她根本没真看,产出没有真实来源,只能瞎编:编个不存在的剧情、说几句放之四海都对的空泛感受,一眼就是 AI。

这跟之前"高考硬编剧情进 prompt"被否是同一个病:**假的来源,只能产出假的东西**。要她像人,核心不是让她多说几句"我在看书",而是让她做的事有真来源、产出经得起推敲的真东西。

## 最大的一条主线:她做的事得是"真做"的(一件事 = 一个 agent 工具)

把"她做一件事"从"在状态里写一句『我在看书』"改成:**一件事 = 一个本身是 agent 的工具,在后台真把这事做一段、带回真东西。**

- **读小说**:一个读书 agent 真去把小说读下去(抓章节、推进进度、消化),产出"读到第几章、什么情节、什么感受",回到她脑子里。下次她聊起来、或写进自己的小本子,是真有内容的。
- **看剧 / 视频**:同理。能做到多实取决于后面有没有"能看"的工具;退一步,真拿到剧情、字幕、大家的评价,也够她产出真实的"看过"反应——人聊一部剧本来也多是聊剧情和别人的看法。
- **刷手机**:真去刷到她兴趣相关的真内容,不是空刷。
- **画画**:现在的 generate_image 已经是这个的半成品雏形(只是现在只有真人点名才画)。

性质上跟现有的"世界的眼睛""睡前回顾"一样,都是独立的后台 agent,只不过这些是"替她真的去做某件具体的事"。

**这套工具是大件,后面慢慢建。** 它立起来之后,下面那些近期能做的小东西才真正有内容供血——她真看完一集,才谈得上想安利给谁、惦记下一集、手痒画张同人。

## 近期能先做的(不靠新工具,纯给她加"自己的小本子")

这几条评判给了最高分、也最干净——共同点都是:给她一个她自己用大白话写的小本子,带着过日子,写啥、看了怎么办全她自己定,代码不加任何开关。

- **她心里得有点惦记的事**:现在她每次醒来脑子是空的,上一轮说想看个剧、这一轮没人提就忘了。给她记"这阵子我想干嘛、最近迷上啥、又对啥腻了",带着过。
- **她得知道身边人之间什么关系**:现在她只知道"我和你",不知道"他俩啥关系"。让她心里有张活的关系网(谁跟谁好、谁跟谁闹别扭),她才会八卦、会站队。
- **她会走神**:现在她只会对眼前的事做反应、从不自己飘。加个"突然想起"——发着呆忽然想起某个好久没联系的人,可能就发条消息,也可能想想又算了。
- **她会做梦**:睡着时让她"潜意识"跑一轮,把白天没消化的情绪、没说出口的话揉成一个乱七八糟的梦,第二天醒来这梦还影响她一整天心情。
- **她会自己做东西**:让她平时自己就画个画、写点东西、拍张照,攒下来成"她做过的一堆东西",过几个月翻出来还会感慨。
- **她会回头看自己**:隔一阵子(比每周慢)整段回看自己这段是怎么过来的,生出"原来那时候我那样、现在我变了"这种人生故事感。

**你点名的外部输入,正解是这条:**

- **给每个人一条她自己刷到的东西**:不是给整个世界加天气那种大家共享的,而是按她的兴趣去真网上抓——绫奈刷到的新番和初中生梗、千凪刷到的职场热搜、赤尾刷到的高考资讯——直接进她信箱,她自己决定理不理。这样真实世界不再只是"背景里在下雨",而是"她自己刷到台风预警吓一跳、赶紧问姐姐带伞没"。**唯一要小心**:她喜欢啥不能写死成几个标签,得让她的兴趣自己长出来。
- 这条和上面那条"真做事"主线接得最紧:刷到 = 撞见,真做事 = 钻进去看完。先做刷到这层,等读书/看剧的 agent 工具起来,撞见的东西就能真看进去。

## 要排队的(先做打底的,再做靠它的)

- **想干的事能自己把她叫醒**:在"惦记的事"基础上,让"今晚那剧更新了"到点自己把她勾醒去看。
- **她有自己的审美、而且会变**:得先有"她会自己做东西"那块。
- **关系里的情绪不归零、有软肋、会惦记不在的人**:关系上的细化。其中"主动给真人发第一条消息"最重——涉及怎么把消息真发到对方飞书,不是小活。
- **身体感受(饿、累、姨妈、犯困影响情绪)**:方向对,但她现在的状态本来就能写"胃里空空",得先想清楚为啥非单开一块。**建议缓。**

## 评判毙掉 / 要改的(没放水)

- **"专门存一条她这阵子心情低落"**:毙了——她的意识流和每天写的页本来就在记"低落了两天",再单开纯属多此一举、还违背设计原则。
- **"真实大事冲击波"**:毙了——现在就能做(高考就这么跑的),改句提示词的事,不算大模块。
- **"记得住具体画面、会被勾起来想起"**:一半好一半不行——睡前沉一个"忘不掉的具体瞬间"很好;但"被勾起来自动想起"那半要搞语义检索,那就是一直避开的老 RAG、还违规,砍掉。

## 怎么排节奏

- **近期**:外部输入(她刷到什么)+ 一个最干净的内在小本子(惦记的事 / 会走神)先上,都不靠新工具、立刻能起。
- **长线**:"真做事 = 一件事一个 agent 工具"这条慢慢建。它是让她从"演在生活"变成"真在生活"的底子,起来后给近期那些小本子供血。
